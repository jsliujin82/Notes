
## Markdown 操作快捷键
| 快捷键 |                                         呈现效果 |
|:-------|-------------------------------------------------:|
| `,n`   |                                              --- |
| `,b`   |                                    文本 **加粗** |
| `,s`   |                                   文本~~中划线~~ |
| `,i`   |                                      文本 *斜体* |
| `,d`   |                                         `代码块` |
| `,c`   |                                      大 `代码块` |
| `,m`   |                                     - [ ] 复选框 |
| `,p`   |                                             图片 |
| `,a`   |                                       [超链接]() |
| `,1`   |                                          # 标题1 |
| `,2`   |                                         ## 标题2 |
| `,3`   |                                        ### 标题3 |
| `,4`   |                                       #### 标题4 |
| `,l`   |                                         -------- |
| `,f`   |                   跳转到至下一个 `<++>` (占位符) |
| `,w`   | 跳转至下一个 `<++>` (占位符) 然后帮你键入`Enter` |
